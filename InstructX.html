InstructX is the most easiest programming language ever!

Step 1: Download InstructX.exe

Step 2: Create a txt file.
Step 3: Write your instructions
Step 4: Open InstructX.exe and type "run --filename--" (e.g "run instruction.txt")
(Note that InstructX.exe and the file you are coding in must be in same directory please.)

This programming language cannot not be used to make applications nor games.
It can only be used to play around with. This is only version 1.0 of InstructX.
I'm still working on version 2, where it'll be a little more better.

Keywords and statements

-Print
-MovePointer (Left/Right)
-Add (integer)
-Subtract (integer)
-PrintBLOCK
-Delete Blocks' Value
I know it's not alot but more will be added in version 2
